Hey, I'm Frank. I love going through the cards of new and older "Magic: The Gathering" sets. I built this site as my ideal card viewing experience.

The background was inspired by one of the newer sets at the time "Unfinity," a humorous set of cards with a space theme.
(The "Mountain" card from the Unfinity set is pictured to the left.)

I enjoy the interesting card abilities, art/style of the cards, and always want an up-close and detailed look at them. To help accomplish this, Cardboard in Space utilizes the Scryfall.com API to fetch the desired card image data from over 20,000 cards.

This site was started in Django using Python, but was changed to improve data fetching speed from the often updated Scryfall.com API.
This project used the following during development:
Javascript, HTML, CSS, Django, Python, Three.js, JQuery, and the Scryfall.com API.
